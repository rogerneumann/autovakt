Vakt — Layout Preview

Reflects Block 12c slot-based rendering. Four layouts: ARC, GRID 2, GRID 4, GRID 2×3. Toggle dark/light theme below. Slot values match the EV demo data used by OBD2Repository.runDemoLoop().

A ARC Layout slot[0] drives arc · slots[1–3] side readouts
Slot 0 → arc gauge (value scaled 0–100). Slot 1 left readout, Slot 2 right readout, Slot 3 footer text. Arc color follows SOC thresholds (cyan >20%, yellow >10%, red ≤10%).
B GRID 2 1×2 · 2 slots · auto-selected on small screens
Two equal tiles stacked vertically. Largest text of any layout — optimised for glanceability at distance. Recommended for small AA head units (<600dp) and compact phones (<400dp).
C GRID 4 2×2 · 4 slots · default for normal screens
Four equal tiles in a 2×2 grid. Default layout for normal-size AA head units (600–900dp) and large phones. Profile seeds these slots from defaultSlots; user can reassign in Settings → Dashboard Layout.
D GRID 2×3 6 slots · opt-in · large screens only
⚠ 6 values while driving — use sparingly. Available as an opt-in in Settings with an inline warning. Auto-selected on very large AA head units (>900dp) and tablets (≥600dp smallestScreenWidthDp). Smaller text than GRID 4.
